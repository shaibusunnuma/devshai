The QuizApp

A simple app to learn geography

This is my first app created with android studio and Kotlin. It is written purely in kotlin and xml.

In this project I focused on linear layouts and how the screen will transition on different divices. As such, all the acitivities were designed with linear layouts. One other thing I learned from this project is desingning a text view to look like a button with button properties. Thanks to android studio's layout properties.

As usual this project was created with love and you can check it out here. You can clone it and modify it anyway you. Please don't hesitate to contact me if you have any questions or ideas to make it better.
